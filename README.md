# VoiceOfSeoul_AnalysisGuide
스마트 인공지능 시티 구축을 위한 도시데이터 표준분석모델 가이드\
-머신러닝 분류기법 활용을 통한, 시민 민원 자동분류-


## 1. 코드 설명
#### 01preprocess
 - 전처리 관련 코드
#### 02word2vec
- word2vec 모델링 및 랜덤포레스트 활용 민원 분류
#### 03stats
- 기술통계량/단어 빈도수 분석
